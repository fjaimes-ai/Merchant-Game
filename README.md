# Merchant Game
**Version 1.0**

---

## Description

Merchant Game is a browser-based trading game. You play as a traveling merchant moving between four towns — Harbormouth, Millfield, Ironhold, and Aldenmoor — buying goods cheaply in one place and selling them for profit in another.

Each town specializes in different goods, so prices vary. Random events on the road and market shortages or gluts keep things unpredictable. Start with 100 gold and see how wealthy you can become.

---

## How to Use

**Starting the game**

1. Download or clone the project folder.
2. Open `index.html` in any modern web browser. No installation or internet connection required.

**Playing the game**

- **Buy goods** — In the Market section, enter a quantity and click "Buy" next to any good. A confirmation box will appear before the purchase goes through.
- **Sell goods** — Enter a quantity and click "Sell" next to any good you're carrying.
- **Travel** — Click any "Travel to..." button to move to a new town. Travel advances the turn and may trigger random events.
- **Prices** — Prices for a good are hidden ("?") until you buy or sell it in that town for the first time.
- **Events** — When traveling, you may hear news of shortages or surpluses affecting prices in a town. Watch the Message Log on the right for updates.

## Troubleshooting

**The game doesn't load**
Make sure you're opening `index.html` directly in a browser, not a text editor. All three files (`index.html`, `styles.css`, `game.js`) must be in the same folder.

**Buttons don't respond**
Try refreshing the page. If the problem persists, open your browser's developer console (F12) and check for error messages.

**Prices all show "?"**
This is expected at the start. Prices are revealed as you trade. Buy or sell a good once to see its price in that town.

**The page looks unstyled**
Check that `styles.css` is in the same folder as `index.html` and that the filename is spelled correctly.
